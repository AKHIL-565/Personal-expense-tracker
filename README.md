# Delivery Money Manager

A clean, mobile-first web app for Swiggy delivery partners to track income, expenses, and COD wallet balance.

## Features
- **Income Tracking**: Add Online or COD income.
- **Expense Tracking**: Categorize expenses (Fuel, Food, etc.).
- **COD Wallet**: Track collected cash vs deposited cash to know your exact payable balance.
- **Filtering**: View data by Today, Week, Month, or All Time.
- **Real-time Summaries**: Net Profit, Total Expense, etc. update instantly.
- **Export**: Download transaction history as CSV.
- **Dark Mode**: Optimized for low-light usage.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the app:
   ```bash
   npm run dev
   ```

3. Open the browser (usually http://localhost:5173).

## Tech Stack
- React
- Vite
- Date-fns
- Lucide React (Icons)
- Vanilla CSS (Mobile-first design)
